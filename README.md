# Upayout Uengine

## Project Overview
This repository showcases QA/SDET work on a sample **Fintech — Payouts & Disbursements** application. It is a portfolio project built with dummy/sample data only — no real client names, production data, or confidential information are included.

## My Role
QA Engineer / SDET — responsible for test planning, manual and automated test coverage, API testing, defect tracking, and test reporting for this project.

## Tech Stack & Tools Used
Playwright, TypeScript, Postman, JMeter, Jenkins CI/CD, Git

## Types of Testing Performed
- Functional Testing
- API Testing
- Regression Testing
- Smoke & Sanity Testing
- End-to-End (E2E) Automation
- Cross-Browser Testing
- Performance/Load Testing (where applicable)

## Key Achievements
- Designed and executed test cases covering core user flows
- Built automated regression suite reducing manual testing effort
- Logged and tracked defects through full bug lifecycle
- Produced test summary reports for each release cycle
